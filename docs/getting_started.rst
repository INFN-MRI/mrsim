Getting Started
===============

Installing MRSim
-----------------

MRSim is available on PyPi

.. code-block:: sh

    pip install mrsim

Development Version
~~~~~~~~~~~~~~~~~~~

If you want to modifiy the mrsim code base

.. code-block:: sh

    git clone https://github.com/INFN-MRI/mrsim
    pip install -e ./mrsim[test, dev, doc]
